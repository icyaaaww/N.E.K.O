# Подключение NEKO к QwenPaw

## Руководство по установке QwenPaw

### Шаг 1. Установка

Python не нужно настраивать вручную. Достаточно одной команды, и установка выполнится автоматически. Скрипт сам загрузит `uv` (менеджер пакетов Python), создаст виртуальное окружение и установит QwenPaw вместе со всеми зависимостями, включая Node.js и фронтенд-ресурсы. Обратите внимание: в некоторых сетевых средах или при корпоративных ограничениях прав это может не сработать.

macOS / Linux:

```bash
curl -fsSL https://qwenpaw.agentscope.io/install.sh -o install-qwenpaw.sh
less install-qwenpaw.sh
bash install-qwenpaw.sh
```

Windows (PowerShell):

```powershell
Invoke-WebRequest https://qwenpaw.agentscope.io/install.ps1 -OutFile install-qwenpaw.ps1
Get-Content .\install-qwenpaw.ps1
powershell -ExecutionPolicy Bypass -File .\install-qwenpaw.ps1
```

### Шаг 2. Инициализация

После завершения установки откройте новый терминал и выполните:

```bash
qwenpaw init --defaults
```

Здесь есть полезное предупреждение о безопасности. QwenPaw прямо сообщает:

> Это персональный помощник, который работает в вашей локальной среде. Он может подключаться к каналам, выполнять команды и вызывать API. Если несколько человек используют один и тот же экземпляр QwenPaw, они будут разделять одинаковые права доступа, включая файлы, команды и секреты.

![Изображение 1 для включения канала Neko](assets/openclaw_guide/image1.png)

Чтобы продолжить, нужно выбрать `yes` и подтвердить, что вы понимаете предупреждение.

### Шаг 3. Запуск

```bash
qwenpaw app
```

Если запуск прошел успешно, в последней строке терминала появится:

```text
INFO:     Uvicorn running on http://127.0.0.1:8088 (Press CTRL+C to quit)
```

После запуска сервиса откройте `http://127.0.0.1:8088`, и вы увидите консоль QwenPaw.

## Настройка канала NEKO: подключение NEKO к QwenPaw

После инициализации QwenPaw автоматически создаст каталог конфигурации. В Windows путь по умолчанию: `C:\Users\ВашеИмяПользователя\.qwenpaw`, в macOS: `~/.qwenpaw`. Все встроенные навыки включаются автоматически.

Найдите этот путь. Папка `.qwenpaw` скрыта, поэтому:

- Пользователям Windows нужно открыть Проводник, выбрать `Вид > Показать` и включить отображение скрытых элементов.
- Пользователям macOS нужно открыть Finder, перейти в домашнюю папку и одновременно нажать `Command + Shift + .`

Скопируйте подготовленный файл конфигурации канала `custom_channels` в папку `.qwenpaw`.

Скопируйте [файлы из папки персонажа](assets/openclaw_guide/%E6%9B%BF%E6%8D%A2%E5%86%85%E5%AE%B9.zip) в `.qwenpaw/workspaces/default`, затем удалите `BOOTSTRAP.md`.

После этого нажмите `CTRL+C` в терминале, чтобы остановить qwenpaw, затем снова запустите его командой `qwenpaw app`.

Далее выполните шаги с изображения, чтобы включить канал Neko.

![Изображение 2 для включения канала Neko](assets/openclaw_guide/image2.png)

## Базовая настройка: конфигурация модели

Нажмите на модель, затем выберите DashScope. В зависимости от вашего API Key можно выбрать и другую модель. Откройте настройки, введите API Key Alibaba Cloud Bailian и сохраните изменения.

![Изображение 3 для включения канала Neko](assets/openclaw_guide/image3.png)

После сохранения вернитесь в чат, и вы сможете выбрать настроенную модель.

Вернитесь в N.E.K.O, и openclaw будет готов к использованию.
